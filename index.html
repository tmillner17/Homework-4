Crystal Collector!

In this game you will be given a number at the beginning of the game.

There are four crystals below that you will click to add an amount of points to your total

The goal of the game is to use the crystals below to equal the displayed number

You win the game if your number matches the number and lose if the number goes over

The value of eacch crystal is hidden until it is clicked on

Each time the game starts the value of each crystal the value of the crystals will change
